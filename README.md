## 👇 &nbsp;Current project

I'm currently working on [@Kogi state Environmental Intelligence Platform](https://kogistateenvironmentplatform.vercel.app/). KSEIP (Kogi State Environmental Intelligence Platform) is a centralized digital platform that provides real-time environmental data, weather insights, disaster monitoring, and geospatial intelligence across Kogi State. It empowers government agencies, emergency responders, and citizens with actionable information to support environmental management, climate resilience, and informed decision-making.
